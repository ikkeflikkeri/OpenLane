# OpenLane

OpenLane is a premium car auction web application built with **SvelteKit**, **TypeScript**, and **Tailwind CSS**. It features a dynamic landing page, an interactive inventory with advanced filters, and detailed auction listing pages with real-time countdown timers, bid history, and a bid submission flow.

## Features

- Modern, responsive UI with Tailwind CSS
- Inventory filtering (status, fuel, transmission, year range)
- Live auction details with countdown and bid history
- Component-based architecture for easy reuse

## Development

```bash
npm install
npm run dev -- --open
```

## Build

```bash
npm run build
npm run preview
```
